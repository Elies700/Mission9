TPS Calculatrice







Ce TP m’a permis d’apprendre à créer une mini-calculatrice web en utilisant HTML, CSS et JavaScript. J’ai appris à manipuler le DOM, gérer les événements, effectuer des calculs simples, tester la parité d’un nombre et permuter des valeurs. J’ai aussi su lier un script JavaScript à une page HTML et ajouter une fonction pour réinitialiser le formulaire. Ce travail m’a aidé à mieux comprendre l’interaction entre le code et l’interface d’une page web.
